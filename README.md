# Bilal Shahzad — Portfolio Website

A dark-theme personal portfolio built with **Angular 17**, showcasing my work as a Business Central Developer, UI/UX Designer, F&O Technical Consultant, Front End Developer, and Odoo Developer.

🌐 **Live:** [Portfolio Website](https://github.com/bilalshahzad996/Portfolio-Website-Angular)

---

## Features

- **Dark theme** design system with purple/cyan accent palette
- **Typewriter animation** cycling through roles in the hero section
- **Scroll-driven BS circle** — rotating animation fixed to the bottom-right corner
- **Animated sections** — elements fade in as you scroll via IntersectionObserver
- **Skill progress bars** with animated fill on scroll
- **Working contact form** powered by [Web3Forms](https://web3forms.com/) — delivers directly to email
- **Custom BS favicon** (SVG + ICO, multi-size)
- **Fully responsive** — tested from 360px to 1440px+
- Google Fonts (Inter + Poppins) and Font Awesome 6.5 icons

## Sections

| Section | Description |
|---|---|
| Hero | Name, animated role typewriter, social links, CTA buttons |
| About | Bio, highlights, tech tags, floating stat cards |
| Skills | Progress ring bars for 7 core technical skills |
| Services | Six service cards (BC Dev, UI/UX, F&O, Front End, Odoo, ERP Integrations) |
| Contact | Contact info + Web3Forms-powered message form |

## Tech Stack

- **Framework:** Angular 17 (standalone components, `@for` control flow)
- **Styling:** SCSS with CSS custom properties
- **Icons:** Font Awesome 6.5
- **Fonts:** Google Fonts — Inter & Poppins
- **Contact Form:** Web3Forms (no backend required)
- **Animations:** IntersectionObserver API + CSS transitions

## Getting Started

```bash
# Install dependencies
npm install

# Start dev server
ng serve

# Open in browser
# http://localhost:4200
```

## Build

```bash
ng build
```

Build artifacts are output to the `dist/` directory.
